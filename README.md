# ANUMATICS

Enterprise Asset & Resource Management System.

## Features

- **Dashboard / Home Screen**: Real-time snapshot of assets, maintenance, and reservations.
- **Organization Setup**: Manage departments, asset categories, and employees.
- **Asset Registry**: Track assets through their full lifecycle.
- **Allocations & Buffer**: Multi-level check to prevent double allocation with integrated return/transfer flow.
- **Resource Booking**: Shared resource scheduler with overlap checks.
- **Maintenance & Audits**: Route repairs through approval and run structured physical audit cycles.
- **Compliance Delta Logs**: Detailed audit trail of all changes.

## Tech Stack

- React 19
- TypeScript
- Vite
- TailwindCSS 3
- Motion (Framer Motion)
- Lucide React
